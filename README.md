Chronos Calendar – Smart Event & Schedule Management System

Overview

Chronos Calendar is a modern and intuitive calendar management application designed to help users organize their schedules, manage events, and improve productivity. The platform provides an efficient way to create, update, track, and manage daily activities through a user-friendly interface.

The application is built to streamline time management by offering event scheduling, reminders, calendar views, and task organization features, making it suitable for students, professionals, and teams.

Features

* Interactive calendar interface
* Create, edit, and delete events
* Daily, weekly, and monthly calendar views
* Event reminders and notifications
* Task and schedule management
* Responsive and user-friendly design
* Secure user authentication
* Real-time event updates
* Search and filter events
* Productivity-focused dashboard

1. User logs into the application.

Installation

Clone the Repository

git clone https://github.com/your-username/chronos-calendar.git

Navigate to the Project Directory

cd chronos-calendar

Create Virtual Environment

python -m venv .venv

Activate Virtual Environment

Windows

.venv\Scripts\activate

Mac/Linux

source .venv/bin/activate

Install Dependencies

pip install -r requirements.txt

Run the Application

python app.py
